# Snack & Stream

Snack & Stream is a movie and TV discovery website designed to help users quickly find something to watch without endless scrolling.

---

## Features/Pages

- **Trending Page** – View popular movies from the TMDB API
- **Search Page** – Search for movies by title
- **Save Movies** – Save favorites to your personal dashboard
- **Dashboard** – View and manage your saved movies
- **Quiz** – Get movie recommendations based on your mood
- **About Page** – Learn about the project and team
- **Contact Page** – Send feedback through a simple form

---

## How It Works

- Movie data is fetched dynamically from the TMDB API
- Users can search for movies or browse trending titles
- Clicking **Save** stores movies in the browser using LocalStorage
- Saved movies are displayed on the Dashboard page
